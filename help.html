✏️ 수정하기 / 🔒 계산하기 버튼
피해율 박스 위 두 개의 버튼으로 입력 상태를 직접 제어합니다.

✏️ 수정하기 — 모든 입력칸이 활성화되어 값을 자유롭게 수정할 수 있습니다.
🔒 계산하기 — 입력칸이 잠겨 실수로 값이 바뀌는 것을 방지합니다.

계산하기 버튼을 누르면 피해율 산정에 영향을 주는 필수 항목(평년수확량·실제경작면적·표본구간 등)의 누락 여부를 자동으로 점검합니다. 미입력 항목이 있으면 ⚠️ 팝업으로 안내되며, 모든 항목이 입력된 경우에만 잠금 모드로 전환됩니다.

모든 입력한 값은 녹색으로 바뀌며 자동으로 실시간 반영되나, 조사 내용 입력이 끝나면 계산하기를 눌러 결과를 확정하고, 수정이 필요할 때는 수정하기를 눌러 잠금을 해제하세요.
